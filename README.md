# Homebrew Tap for Pasta

This is the official [Homebrew](https://brew.sh/) tap for [Pasta](https://pasta-app.com), a blazing-fast clipboard history manager for macOS.

## Installation

```bash
brew tap crmitchelmore/pasta
brew install --cask pasta
```

## Updating

```bash
brew upgrade --cask pasta
```

## Uninstalling

```bash
brew uninstall --cask pasta
brew untap crmitchelmore/pasta  # optional
```
